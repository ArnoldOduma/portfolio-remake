Eazy Search

Eazy search is an application that gets your location and lists businesses, services and housing near you. All the businesses, housing and services are authenticated first before posting. A user can also search the application for services near them and be able to view the contact details of the service provider. It was made by a team of two. I did the front end and James Mutahi did the Django API Backend

Emiwell

The application is also connected to the counsellors callendar so that incase of a booking the calendar is automatically updated.
